User Story: 

As an Admin User, I want to reset my password, so that I can regain access to my account if I have forgotten my password.

Acceptance Criteria:
 
 - The Admin User can access a ""Forgot Password"" page on the Web Portal or app.
 - The system prompts the Admin User to provide their username.
 - The system validates the provided username.
 - System displays appropriate error message for invalid username
 - The system sends an OTP (4 digit) to the Admin User's registered Mobile number.
 - The Admin User can enter the the OTP to initiate the password reset process.
 - The Admin User has the option to resend OTP in case user doesnt recieve within configurable 60 seconds.
 - The system displays appropriate error message if the entered OTP is invalid.
 - The system prompts the Admin User to create a new password upon successful verification of the OTP.
 - The new password should be validated with Confirm password. 
 - The system displays appropriate error message if the new and Confirm password mismatch.
 - The system validates the new password against password policy requirements (e.g., minimum length, complexity).
 - The system updates the Admin User's current password with the new password.
 - The system provides a confirmation message upon successful password reset.
 - After successful password reset, user will be navigated to login screen
 - The Admin User can log in to the web portal and app using their new password."
